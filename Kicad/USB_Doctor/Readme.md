This is a basic usb doctor/charging monitor that measures and displays the voltage, current and power that is being drawn through, It consits of a male USB A connector on one end which can
be connected to adapters and the device has a female USB A connector on other end to which the device to be charged can be plugged in.

## Components:
  1. Atmega328p
  2. Ina219 sensor
  3. 10k resistors
  4. ams1117 3.3v voltage regulator
  5. usb connectors
  6. capacitor - 100nf
